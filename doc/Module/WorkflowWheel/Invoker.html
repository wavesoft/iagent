NAME

Module::WorkflowWheel::Listener - The listening sub-module of a workflow wheel

DESCRIPTION

This sub-module implements the incoming message handling for the workflow system. It monitors the pub/sub channel for action queries, checks availability and status of each action and performs the first-level filtering for the incoming messages.
