NAME

Module::XMPPCLI - Simple XMPP Bridge to Command-Line interface

DESCRIPTION

This module provides xmpp access to the locally provided command line. This module requires the iAgent::Module::CLI to be loaded in order to work properly.

KNOWN ISSUES

Since the CLI system does not support concurrent users, every time a ueser types something on CLI he will get the output redirected to him.
