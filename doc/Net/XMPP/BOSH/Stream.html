NAME

Net::XMPP::BOSH::Stream - BOSH XML Stream

SYNOPSIS

Net::XMPP::BOSH::Stream is an extension to XML::Stream that wraps the standard XMPP XML messages to BOSH.

For more usage details see Net::XMPP::Client

DESCRIPTION

This module re-blesses the XML::Stream object inherited from the Connection with the Net::XMPP::BOSH::Stream that utilizes the BOSH protocol.

It is not intended to be used directly. It's a private package used by the Net::XMPP::BOSH::Client.
