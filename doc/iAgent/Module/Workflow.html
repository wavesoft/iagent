Workflow Pubsub Object

Here is a short description of the XML Namespace "iagent:workflow", wich is used by iAgent Workflow plugin to implement the workflow system

Information Query

An important part of this system is the information discovery status query. The general syntax of the information query node follows:

  <item xmlns="iagent:workflow">
    <query for="...">
        <!-- Condition -->
        <cond name="..." value="..." />
        <!-- Parameter -->
        <parm name="..." value="..." />
        
    </query>
  </ite<>

List Actions

List root actions:

  <item xmlns="iagent:workflow">
    <query for="actions">
        <cond name="type" value="root" />
    </query>
  </ite<>

List all actions:

  <item xmlns="iagent:workflow">
    <query for="actions">
    </query>
  </ite<>

List active:

  <item xmlns="iagent:workflow">
    <query for="actions">
        <cond name="type" value="root" />
    </query>
  </ite<>
