NAME

iAgent::Module::XMPP::MUC - Multi-User Chat implementation for the XMPP Module

DESCRIPTION

This XMPP Plugin implpements the XEP-0045 (Multi-User Chat) standard and exposes it's functionality through the comm_group_* messages.

HANDLED MESSAGES

comm_group_join HASHREF

Join the specified MUC channel

comm_group_leave HASHREF

Leave the specified MUC channel

comm_group_send HASHREF

Send a message to the specified MUC Channel

comm_group_members HASHREF

List the members of the specified MUC Channel

BROADCASTED MESSAGES

comm_group_chat HASHREF

A mesage has arrived

comm_group_joined HASHREF

A user has joined the channel

comm_group_leaved HASHREF

A user has left the channel
